# README
コンセプト：人の考えを知ると同時に、自分の考えも知ってもらうきっかけをサポートする
取り入れた技術：⑴　LINE、Google、Facebookを利用してのログイン方法の実装
            ⑵　非同期でのチャット機能の実装
            ⑶　チャットに禁止ワードを設定（死ね、バカ等）
            
This is application that can chat with asynchronous bidirectional communication.
